# Rust Card Deck

A simple Rust project focused on learning the fundamentals of the language.

## 🚧 Status: Work in Progress
This project is currently under active development as part of my journey through the **Stephen Grider Rust Course**. I am building this feature by feature as I learn.

## Current Features
- Custom `Deck` struct to hold card data.
- Automated card generation using nested `for` loops.
- Use of the `format!` macro to build card name strings.
- Vector management (`Vec<String>`) and basic mutability.
